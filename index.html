Технологии, используемые при создании сайтов

HTML

Cтандартизированный язык разметки веб-страниц во Всемирной паутине. То, что видит пользователь на странице - и есть HTML код

CSS

CSS создан для позиционирования элементов на странице, для придания им определённого задачей вида.

JavaScript

Используется для взаимодействия с серверами и базами данных, для создания динамических web-страниц

PHP

язык программирования, используемый как встраиваемый язык для программного доступа к объектам приложений, но наиболее широкое применение находит как язык сценариев для придания интерактивности веб-страницам.

CMS

Используемая для обеспечения и организации совместного процесса создания, редактирования и управления содержимым сайта.

MySQL

С помощью этой системы создаются и управляются базы данных, хранящие информацию о сайте

Структура и содержание web-сайтов

Структурой интернет-ресурса называют схему расположения его основных элементов, то есть страниц, записей, разделов относительно друг друга. НО она отражает не только строение сайта, но и логическую связку всех его составляющих.

Виды внутренних структур сайтов

Линейная

Логика такой структуры – ознакомить пользователей сайта со всеми его страницами, расположенными в определенной последовательности. Линейная схема применяется в сайтах-презентациях и портфолио, где нужно постепенно ознакомить посетителя с наиболее важной информацией.

Линейная с ответвлениями

Благодаря ответвлениям на одном сайте можно показывать сразу несколько продуктов. Переходя на ветку, пользователь будет видеть постраничную презентацию. Но имеющую несколько версий или вариаций.

Древовидная

И наиболее универсальный вариант. В нем каждому товару или услуге отводится отдельная ветка: раздел или подраздел. Привычнее всего пользователям общаться именно с такими ресурсами.

PHP

PHP был разработан датским программистом Расмусом Лердорфом в 1995 году. PHP применяется для работы с серверами и базами данных, то есть работает в области back-end, в отличии от JS PHP более приспособлен для этого и ему не требуется дополнительные платформы или модули.

Задачи, решаемые PHP:

  • Отправка форм

    С помощью PHP можно отправлять данные с форм, заполненных пользователем

  • Взаимодействие с базами данных

    К сайту на PHP подключается база данных. Это во многом упрощает разработку и обеспечивает большую безопасность, так как вся информация будет храниться в таблицах, а не в самом коде.

  • Использование сессий и cookies

    Они нужны для того, чтобы не забивать базу данных лишними элементами, а хранить часть информации на компьютере пользователя.

  • Загрузка и обработка файлов

    Средствами PHP можно осуществлять редактирование файлов, загрузку и отправку

  • Создание изображений

    PHP также позволяет работать с изображениями, редактировать их, сжимать

Размещение в Интернете

Хостинг — услуга заключающаяся в размещении информационных ресурсов пользователей на серверах хостинг-провайдера.

Домен или доменное имя - это адрес по которому будет доступен сайт.

После регистрации доменного имени и выбора хоста можно загружать файлы на сервер, предоставляемый хостом, на зарегистрированное доменное имя, импортировать базу данных, к которой прикреплен сайт, и он будет доступен в Интернете по зарегистрированному домену. Так делают со всеми коммерческими и деловыми сайтами, но если сайт не задумывался как коммерческий, то нет необходимости платить за хостинг и домен, потому что существуют бесплатные хосты, они, конечно либо значительно уступают платным в производительности, либо имеют какие-либо ограничения для пользователей, например по количеству максимально возможных сайтов для размещения, также большинство из них предоставляю собственный домен, в результате чего сайт будет размещен под доменом хоста, что может быть совершенно не удобно. Более они ничем не отличаются от платных хостов.

GitHub — крупнейший веб-сервис для хостинга IT-проектов и их совместной разработки. Веб-сервис основан на системе контроля версий Git и разработан на Ruby on Rails и Erlang компанией GitHub, Inc (ранее Logical Awesome). Сервис бесплатен для проектов с открытым исходным кодом и (с 2019 года) небольших частных проектов, предоставляя им все возможности для размещения на собственных серверах.
